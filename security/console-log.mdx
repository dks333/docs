## What This Test Does

This console logging test provides:

1. **Basic Console Methods** - Tests availability of log, warn, error, info, debug, trace
2. **Context Information** - Displays current iframe context and environment details
3. **Console Grouping** - Demonstrates organized logging with groups
4. **Performance Metrics** - Shows timing information for debugging
5. **Console Table** - Displays structured data in table format
6. **Persistent Logging** - Sets up ongoing log entries for monitoring

## Purpose

This foundational test helps with:

- **Debugging other security tests** by ensuring console access works
- **Monitoring iframe behavior** through structured logging
- **Performance analysis** of iframe loading and execution
- **Context verification** to confirm iframe environment

## Expected Output

You should see in the browser console:

- Organized log entries with icons and structure
- Context information about the iframe environment
- Performance timing data
- Periodic log entries over time
- Confirmation that logging works properly

This test serves as a baseline to ensure that more complex security tests can properly log their results and findings.
